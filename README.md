# Destiny International Christian Assembly — Official Website

The official website for **Destiny International Christian Assembly (DICA)**, a church based in Birmingham, UK. Built with React and Vite.

## 🌐 Live Site
> Coming soon

## 🛠️ Tech Stack

- **React** — UI framework
- **Vite** — build tool and dev server
- **React Router** — client-side routing
- **CSS Modules** — component-scoped styling
